# Equiviva\n\nAn equestrian management app built in React, implementing the Equiviva design system from Pencil.\n\n## Features\n\n- **Home Dashboard** - Quick overview of upcoming tasks, health metrics, and recent expenses\n- **Expenses** - Track monthly spending broken down by category\n- **Health** - Complete health history and upcoming veterinary appointments\n- **Calendar** - Day view with hourly schedule timeline\n- **Profile** - User profile and horse management\n- **Interactive Navigation** - Bottom tab bar for seamless screen transitions\n\n## Design System\n\nImplements the complete Equiviva design system with:\n\n- Color palette (Sand, Charcoal, Taupe, Burgundy, and category-specific colors)\n- Typography (Cormorant Garamond, Outfit, JetBrains Mono)\n- Component library (Cards, buttons, pills, transaction lists)\n- Responsive mobile-first design\n\n## Getting Started\n\n### Installation\n\n```bash\nnpm install\n```\n\n### Development\n\n```bash\nnpm run dev\n```\n\nThe app will open at `http://localhost:5173`\n\n### Build\n\n```bash\nnpm run build\n```\n\n## Technology Stack\n\n- React 18\n- Vite\n- CSS3\n- Lucide React Icons\n- Google Fonts (Cormorant Garamond, Outfit, JetBrains Mono)\n\n## Architecture\n\n```\nsrc/\n├── components/\n│   └── BottomNav.jsx         # Interactive navigation\n├── screens/\n│   ├── Home.jsx              # Dashboard\n│   ├── Expenses.jsx          # Expense tracking\n│   ├── Health.jsx            # Health records\n│   ├── Calendar.jsx          # Schedule timeline\n│   └── Profile.jsx           # User profile\n├── styles/\n│   └── designSystem.js       # Design tokens\n├── App.jsx                   # Main app component\n└── main.jsx                  # Entry point\n```\n\n## Interactive Elements\n\nOnly the bottom navigation bar is interactive:\n- Click tabs to switch between screens\n- The FAB (+) button is styled but non-functional\n- All other elements are static\n\n## Design Notes\n\nThis implementation faithfully reproduces the Equiviva design from the Pencil `.pen` file, including:\n- Mobile-first responsive design (428px width)\n- All color values from the design system\n- Typography scale and font families\n- Spacing and component layouts\n- Visual hierarchy and shadows\n"
